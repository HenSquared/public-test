# GitHub Copilot: VSCode vs Dotcom Comparison

## Overview

GitHub Copilot is available in both **VSCode** and on **GitHub.com (Dotcom)**, each offering slightly different experiences.

## VSCode

- Integrated directly into the editor via the GitHub Copilot extension
- Uses Bing search to fetch latest news and information
- Displays results as a list of references followed by content
- Inline chat and sidebar chat support

## Dotcom (GitHub.com)

- Accessible via the Copilot icon on GitHub.com
- Clearly labels **References** (sources used) and **Content** (generated response)
- Web-based interface, no installation required

## Key Difference

| Feature       | VSCode                  | Dotcom                  |
|---------------|-------------------------|-------------------------|
| References    | Shown as search links   | Clearly labeled section |
| Content       | Follows references      | Clearly labeled section |
| Access        | Extension required      | Browser-based           |

Both platforms provide powerful AI assistance tailored to their respective contexts.
